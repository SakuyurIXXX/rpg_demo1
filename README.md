### 战斗系统

传统RPG系统，攻击方面拥有三种属性伤害（火、冰、雷）和可造成暴击的物理伤害，其中属性伤害能造成对应的属性异常效果，获取counter技能后可弹反敌人的一些招式。防御方面有护甲值、法抗、HP值、闪避以及可获得的格挡技能和冲刺技能
